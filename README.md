# SQL Injection Detection Tool

## Project Overview

This project is a **SQL Injection Detection Tool** developed in Python. 
It performs both:

- Manual SQL Injection Testing
- Automated SQL Injection Testing (SQLmap)

The tool provides:

- GUI Interface
- Manual Payload Testing
- Automated Detection
- Vulnerability Severity Level
- Scan History
- PDF Report Generation

This project was developed as a **CyberSecurity Project**.

---

## Features

✔ Manual SQL Injection Detection 
✔ Automated SQLmap Detection 
✔ Vulnerability Severity Calculation 
✔ Scan Time Measurement 
✔ Scan History Storage 
✔ Professional GUI 
✔ PDF Report Generation 
✔ Show PDF Button 
✔ Ethical Target Restriction
